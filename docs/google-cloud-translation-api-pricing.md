# Google Cloud Translation API Pricing

## Free Tier

- The Google Cloud Translation API offers a permanent Free Tier of **500,000 characters per month** at no cost.
- This quota automatically renews every month.
- No annual free quota is provided; the limit is strictly monthly.
- The Free Tier applies to translated text characters (including spaces, excluding formatting like HTML tags).

## Pricing After Free Tier

- After exceeding 500,000 characters in a month, usage is billed at approximately **$20 USD per 1 million characters** (as of 2025).
- Charges are calculated per translated character.

## Official Documentation

- [Google Cloud Translation Pricing (including Free Tier)](https://cloud.google.com/translate/pricing)
- [CostGoat Guide & Price Calculator](https://costgoat.com/pricing/google-translate)

## Summary

- Every Google Cloud account receives 500,000 translated characters per month for free.
- No special setup or trial is required for the Free Tier.
- After the free quota, standard rates apply.
- For up-to-date pricing and details, always refer to the [official documentation](https://cloud.google.com/translate/pricing).
