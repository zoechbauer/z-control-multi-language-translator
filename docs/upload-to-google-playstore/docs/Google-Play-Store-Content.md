### Full Description EN (4000 characters max)

```
z-control Translator helps you learn and refresh foreign languages simultaneously — perfect for preparing your next holiday! Powered by Google Translate, see translations side-by-side in up to 5 languages and compare how different languages express the same idea.

🌍 FEATURES:
✓ Translate up to 100 characters into up to 5 languages simultaneously
✓ Listen to translations with text-to-speech
✓ Adjust speech speed and pitch on mobile devices
✓ Voice input via your device keyboard
✓ Light & dark mode (follows your system settings)
✓ Bilingual interface: English & German
✓ Works on mobile, tablet, and web browsers
✓ Free with a shared monthly quota system

👥 PERFECT FOR:
• Travelers & language learners
• Professionals & communicators
• Anyone needing quick, reliable translations

🛡️ PRIVACY & SECURITY:
✓ Your text is NOT stored by this app
✓ Translations are processed by Google's secure servers
✓ No data collection or tracking
✓ 100% private and secure

📊 QUOTA SYSTEM:
Use of Google Translate API is free up to a shared monthly limit (500,000 characters for all users and 10,000 characters per user). Text-to-speech is unlimited and free. The app notifies you before the limit and uses simulated translations once quota is reached.

🌐 AVAILABLE EVERYWHERE:
• Download the native mobile app
• Use the web version on any device: z-control-translator.web.app
• Same features, same privacy protection

⚙️ HOW IT WORKS:
1. Select up to 5 target languages
2. Enter text (up to 100 characters)
3. Click "Translate"
4. See translations instantly
5. Click the speaker icon to listen
6. Repeat translations as needed or change target languages!

Supports 100+ languages with automatic language detection. Perfect for travelers, students, and professionals!

Questions? Send us feedback through the app. We're here to help!
```
