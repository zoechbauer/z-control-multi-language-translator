🎯 Exact Size Template - Multiple Options!

📱 Option 1: Direct Screenshot (NO RESIZING!)

  1. Open this HTML file in your browser
  2. Use Snipping Tool (Windows + Shift + S)
  3. Capture ONLY the green area (inside the graphic)
  4. Save as PNG - it's exactly 1024×500!
  5. Upload directly - no Paint needed!

🎨 Option 2: Use Paint 3D (Better Resize)

  1. Take screenshot of green area
  2. Open Paint 3D (free with Windows)
  3. File → Open your screenshot
  4. Canvas → Resize Canvas
  5. Set: Width 1024, Height 500
  6. Click Done
  7. Export as PNG

🌐 Option 3: Free Online Tool

  1. Take screenshot
  2. Go to: resize-image.net or iloveimg.com
  3. Upload your image
  4. Resize to 1024×500 pixels
  5. Download and use

  • Option 1: No resizing = perfect centering
  • Option 2: Paint 3D has better resize controls
  • Option 3: Online tools force exact dimensions
